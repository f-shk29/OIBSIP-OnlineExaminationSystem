# OIBSIP-OnlineExaminationSystem

This is a simple console-based Online Examination System developed using Core Java. The project allows a user to log in with a username and password. After successful login, the user is presented with a set of multiple-choice questions. The user answers by selecting options (a/b/c/d), and the system automatically calculates the score along with performance evaluation.
